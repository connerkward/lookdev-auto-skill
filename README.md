# lookdev-auto

**lookdev-auto** tunes a visual parameter automatically — render labeled variants, have a vision/video model rate them and suggest better values, re-render, repeat. The **automated counterpart to lookdev**: a model judges instead of you.

> **What this is:** Structures an existing Claude ability (look-at-output-and-iterate) into a disciplined loop — not a new technique, a repeatable harness around one.

## Install (Claude Code plugin)

```
/plugin marketplace add connerkward/connerkward-skills
/plugin install lookdev-auto@connerkward-skills
```

Or drop this repo's `SKILL.md` into your agent's skills directory.
